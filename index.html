Introduction

HistoBot is a Discord bot built with JavaScript and DiscordJS library. It's name come from History and Bot. As it name suggest, this bot is a history-themed bot. You can invite the Bot to your server with this following link:

https://discord.com/api/oauth2/authorize?client_id=776592026799571014&permissions=523328&scope=bot

HistoBot Purpose's

HistoBot purpose is to provide daily history knowledge, like events, historical figure births and deaths. Aside from that, Histobot also have fun and misc commands from meme to COVID-19 cases information

DiscordJS

As mentioned before, this bot is made with the help of DiscordJS library. Discord.js is a powerful node.js module that allows us to interact with the Discord API very easily. It takes a much more object-oriented approach than most other JS Discord libraries, making our bot's code significantly tidier and easier to comprehend. Read more about DiscordJS:

https://discord.js.org/

Bot Prefix

Prefix is a symbol or character that attached before command word. It helps the bot to recognize which message is a bot command and which one is not. For HistoBot, the command prefix is |

The today- Commands

The today- commands consists of the following command type:

  • today-events
  • today-births
  • today-deaths

The commands return a Discord message embed that contains information about important events, births and deaths that happened today. Syntax :

|today-{type}

Example:

|today-events

The search- Commands

The search- commands basically has the same function as today- commands. The difference is that search- commands returns a message embed that contains important information by the entered date and month arguments. It's also consists of the following commands:

  • search-events
  • search-births
  • search-deaths

The command syntax :

|search-{type} {date} {month}.

For example:

|search-births 16 5

history-meme Command

Just like the command name, this one gives user a history meme. The source of the meme is from a Reddit /r/HistoryMemes. The command syntax is simply:

|history-meme

joke Command

This joke command takes 1 argument for the type of joke. Jokes is requested from 15Dkatz/official_joke_api. There are 3 types of joke, general programming and knock-knock. The command syntax is:

|joke {type}

But you can get random jokes from all category by simply send :

|joke

gulag Command

Use this command if you want to send someone to Gulag (just for fun of course). It takes 2 arguments, user mention and reason. Syntax:

|gulag @mention {reason}

Example

|gulag @HistoBot he's a western spy

medal Command

You want to pay respect to your friend for bravery or heroic deeds? Use |medal command! It takes 2 argument, @mention(or random name) and type. As for the type, HistoBot currently has 4 medal types, iron-cross,red-star,victoria-cross and medal-of-honor. Syntax:

|medal @mention {type}

But you can replace the mention with random name. If the name is two words or longer, replace all the spaces with "-". Example:

|medal trupper-hans iron-cross

You also can give medal to yourself by replacing the name or the mention with "me":

|medal me red-star

say Command

You can make the bot say something with this command. It takes words to say as the command argument. Syntax:

|say {words}

Example:

|say your kingdom will come!

shout Command

Same as the say command, but this time all words turns into uppercase (yeah a bit useless since user could use caps lock on previous command). Syntax:

|shout {words}

palindrome Command

This is a palindrome checker. It'll check the word given as argument by user if it's a palindrome or not. Syntax:

|palindrome {words}

Example:

|palindrome A Man, A Plan, A Canal. Panama

romanum Command

Romanum command converts decimal number to roman numerals. Syntax:

|romanum {number}

Example:

|romanum 235

caesar-ciphers Command

The Caesar cipher is named after Julius Caesar, who, according to Suetonius, used it with a shift of three (A becoming D when encrypting, and D becoming A when decrypting) to protect messages of military significance. While Caesar's was the first recorded use of this scheme, other substitution ciphers are known to have been used earlier (Source:Wikipedia).While Caesar originally use shift of three, HistoBot use shift of 13 similar to ROT13 system to encrypt to message. Use this command if you want to send a secret message to your crush XD. Syntax:

|caesar-ciphers {words}

Example:

|caesar-ciphers ATTACK ON TITAN

announce Command

This command is used to send an announcement. It only takes one argument. Syntax:

|announce {message}

Example:

|announce 2020 2 is confirmed!

ping Command

This command is used to check bot latency

|ping

bug-report Command

You find a bug? You can help the Developer squashing it with this command. It'll send your report straight into the Dev DM (so don't spam or you're going to be blacklisted). Syntax:

|bug-report {message}

Example:

|bug-report The bot doesn't reply

covid-country Command

As an effort to support the war against COVID-19, HistoBot provides covid-country where users can request information about COVID-19 cases in specified country so users could keep up with the track. Syntax:

|covid-country {country-slug}

Hint : You can find your country request slug by just simply write |covid-country without any argument.

covid-global Command

This command gives the global information of COVID-19 cases.

|covid-global
